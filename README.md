# Smart Fit 

Smart Fit is a modern, responsive web application designed to help users kickstart their fitness journey. It provides an intuitive interface for account management, administrative tracking, and personalized health analysis.

## Features
* **User Authentication:** Dedicated interfaces for user registration and login.
* **Admin Portal:** Separate administrative access for managing fitness data.
* **Modern Hero Section:** Engaging landing page with interactive call-to-actions and dynamic statistic tracking (Personalized Plans, 24/7 Tracking, BMI Health Analysis).
* **Fully Responsive:** Clean, modern layout built using semantic HTML5 and CSS3.

## Tech Stack
* HTML5
* CSS3 (External Stylesheet)
* Unsplash API (Visual Assets)
